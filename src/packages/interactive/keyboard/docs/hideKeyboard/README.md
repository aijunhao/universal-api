## 安装

```bash
$ npm install @uni/keyboard --save
```

## 示例

```js
import keyboard from '@uni/keyboard';

keyboard.hide();

```

## 方法

### `hide()`

隐藏键盘。

#### 参数

| 成员 | 类型 | 描述 | 必选 | 默认值 |
| --- | --- | --- | --- | --- |
| options         | `object`  | hide 参数 | 否 | - |
| options.success | `Function`  | 成功的回调 | 否 | - |
| options.fail | `Function`  | 失败的回调 | 否 | - |
| options.complete | `Function`  | 结束的回调 | 否 | - |
